### 📻 7. TRADITIONAL / CLASSIC JAZZ
> *"The Purist's Corner"*

```
┌─────────────────────────────────────────────────────┐
│  DEFINISI                                           │
│  Jazz klasik dalam format autentik:                │
│  Bebop, Swing, Cool Jazz, Hard Bop, Modal.        │
│  Artis: Coltrane, Miles Davis, Ella, Sinatra,     │
│  Charlie Parker, Bill Evans, Duke Ellington        │
│  MASALAH BESAR: Copyright! Semua musik ini         │
│  dimiliki label besar.                              │
└─────────────────────────────────────────────────────┘
```

**⚠️ PERINGATAN BESAR:**
```
HAMPIR SEMUA REKAMAN CLASSIC JAZZ MEMILIKI COPYRIGHT
→ Video akan di-claim oleh Sony, Universal, Blue Note Records
→ Revenue dibagi atau dihapus
→ SANGAT BERISIKO untuk channel baru

SOLUSI untuk tetap bisa main di niche ini:
→ Buat musik ORIGINAL bergaya classic jazz
→ Gunakan public domain recordings (sebelum 1928 di AS)
→ Partner program dengan label (advanced)
```

## ❓ PERTANYAAN KUNCI SEBELUM PILIH SUB-NICHE

```
1. Mau fokus ke REVENUE (RPM tinggi)?
   → Pilih Smooth Jazz / Cafe Jazz

2. Mau fokus ke VIEWS (traffic tingi)?
   → Pilih Lo-fi Jazz (tapi persaingan keras)

3. Mau PELUANG niche yang masih terbuka?
   → Pilih Bossa Nova atau Jazz for Sleep

4. Sudah punya AI music tool (Suno)?
   → Semua bisa dikerjakan kecuali Traditional Jazz (copyright)

5. Mau 1 video = 1 upload yang panjang?
   → Sleep Jazz (8 jam) atau Smooth Jazz (2 jam)

REKOMENDASI FINAL UNTUK PEMULA:
✅ BOSSA NOVA — paling seimbang antara competition level,
   RPM, audience quality, dan uniqueness potential
```
